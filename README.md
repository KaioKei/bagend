# BAGEND

Install tools and feel like home with your Debian-based environment.

Install requirements and follow the instructions !


## REQUIREMENTS

* Connected as sudoer
* Run :

```sh
./requirements.sh
```

## INSTALL

In this order :

* [Git](git/README.md)
* GCC C compiler
* [Pyenv](pyenv/README.md)
* [Fish](fish/README.md)
* [Neovim](neovim/README.md)
* [Tmux](tmux/README.md)

  
## OTHERS

```sh
sudo apt install xclip
```
